# android-rick-morty

Android App that consumes a Rick and Morty API for demonstration purposes of MVVM Architecture with
Clean Code.

## Architecture

![Architecture of app](/docs/architecture.png)

## Programming Language

- Kotlin

## Libraries

- Coil
- Hilt
- Jetpack Compose
- Lottie
- Material Design 3
- MockK
- Navigation
- Retrofit

## Features

- Image cache
- Error handling
- Filtering
